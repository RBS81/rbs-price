Инструкция — Google Apps Script
  1. Открой sheets.google.com → создай таблицу RBS История цен
  2. Меню: Расширения → Apps Script
  3. Удали весь код, вставь содержимое файла rbs_google_script.gs
  4. Нажми Развернуть → Новое развертывание → тип «Веб-приложение» → доступ «Все» → скопируй URL
  5. Вставь URL выше и нажми «Сохранить»
